GETTING STARTED
================================================================================

1. Install dependencies:
   $ cd src/portalapp/app
   $ npm install

2. Set up environment:
   $ cp .env.example .env
   (Edit .env with your values)

3. Start development:
   $ npm run dev

4. Open browser:
   http://localhost:3000

5. Read documentation:
   - QUICKSTART.md for fast track
   - README.md for overview
   - ARCHITECTURE.md for technical details
   - DESIGN_SYSTEM.md for components
   - IMPLEMENTATION_STATUS.md for roadmap
